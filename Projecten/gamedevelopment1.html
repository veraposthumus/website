Game development 1

Eisen

Er waren twee eisen aan dit project gebonden:
- Het moet kunnen werken op hele simpele computers
- Er is niet heel veel tijd

Dit zorgde ervoor dat we een redelijk simpel spel moesten maken.

Het plan en wat er mis ging

Na wat brainstormen kwamen we met het idee om ons spel een Mario thema te geven, maar dan met wat aanpassingen.
Dit koste echter iets wat meer tijd dan gehoopt. Deze tijdsnood was vooral veroorzaakt door ons niet al te slimme plan.
Het plan hield in om de code van een spel zo te veranderen dat het ons spel werd. Dit zorgde er alleen voor dat onze code niet langer leesbaar was, het was spagetti code geworden
